# Temporal, connectivity and event feature outputs

This folder contains representative outputs from the temporal, connectivity and event-level EEG analysis notebook.

## Contents

- `figures/` — representative figures generated by the notebook.
- `tables/` — aggregate, statistical and summary tables that can be shared.
- `tables_anonymized_samples/` — small anonymized samples of feature tables.

## Analysis blocks

The notebook includes:

1. DFA/LRTC-like exploratory amplitude-envelope features;
2. burst metrics of alpha and beta amplitude envelopes;
3. functional connectivity features;
4. ROI×ROI connectivity matrices;
5. graph/network metrics;
6. transient event morphology and spatial spread features;
7. subject-level statistical analysis with FDR correction.

## Important notes

Raw EEG data are not included.

Full record-level and subject-level feature tables are not included because they may contain internal subject or record identifiers. Only aggregate statistics and anonymized table samples are provided.

The main statistical interpretation should be based on subject-level aggregation to avoid pseudo-replication when one subject has multiple recordings.

DFA/LRTC-like features should be interpreted cautiously if calculated on short 4-second epochs. They are treated as exploratory fluctuation features rather than strict long-range temporal correlation estimates.
