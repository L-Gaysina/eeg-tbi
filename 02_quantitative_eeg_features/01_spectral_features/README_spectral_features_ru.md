# Спектральный анализ ЭЭГ

Этот ноутбук выполняет расчёт спектральных признаков ЭЭГ и межгрупповое сравнение записей пациентов с черепно-мозговой травмой и контрольной группы.

Ноутбук предназначен для запуска **после этапа предобработки**. На вход подаются очищенные данные после финального контроля качества. На выходе формируются таблицы спектральных признаков, статистические результаты и готовые визуализации.

## Ноутбук

```text
02_spectral_features_v2_topomaps.ipynb
```

Основной входной файл:

```text
work-3/output/tables/analysis_ready_inventory_cleaned.csv
```

## Основные этапы анализа

### 1. Загрузка и проверка данных

В начале ноутбука:

- задаются пути к рабочей папке;
- создаются папки для таблиц, рисунков, логов и кэша;
- загружается очищенный `analysis_ready_inventory_cleaned.csv`;
- проверяется состав выборки после очистки.

### 2. Расчёт PSD

Спектральная плотность мощности рассчитывается методом **multitaper**.

Используемые параметры:

```text
method = "multitaper"
fmin = 1.0
fmax = 45.0
bandwidth = 2.0
adaptive = True
low_bias = True
normalization = "full"
```

PSD рассчитывается в диапазоне **1–45 Гц**.

### 3. Частотные диапазоны

В анализ включены стандартные диапазоны ЭЭГ:

```text
delta: 1–4 Гц
theta: 4–8 Гц
alpha: 8–13 Гц
beta: 13–30 Гц
gamma: 30–45 Гц
```

Для каждого диапазона рассчитываются абсолютная и относительная мощность.

### 4. ROI-схема

Признаки рассчитываются глобально и по областям интереса:

```text
frontal   — лобная область
central   — центральная область
temporal  — височная область
parietal  — теменная область
occipital — затылочная область
```

### 5. Индексы спектрального замедления

Дополнительно рассчитываются slow/fast индексы:

```text
(delta + theta) / alpha
(delta + theta) / beta
log((delta + theta) / (alpha + beta))
```

Логарифмический индекс используется как более устойчивый вариант, поскольку простые отношения могут быть чувствительны к малым значениям знаменателя.

### 6. Характеристики формы спектра

В ноутбуке рассчитываются интегральные характеристики формы спектра:

```text
SEF50
SEF95
нормированная спектральная энтропия
спектральная плоскостность
```

Эти признаки позволяют оценить смещение «центра тяжести» спектра и общую структуру распределения мощности по частотам.

### 7. Апериодическая компонента, alpha peak и residual spectrum

В ноутбуке также оцениваются:

```text
апериодический показатель 1/f
апериодический offset
частота alpha-пика
амплитуда alpha-пика
residual spectrum после удаления 1/f-компоненты
```

Апериодическая компонента оценивается как линейная аппроксимация PSD в log-log пространстве. Это не является полноценным FOOOF/specparam-разложением, если соответствующий пакет явно не используется.

### 8. Статистический анализ

Основная статистическая интерпретация проводится на уровне субъектов.

Это важно, потому что у одного субъекта может быть несколько записей. Если анализировать все записи как независимые наблюдения, можно завысить эффективный размер выборки.

Основной принцип:

```python
roi_features_subject = (
    roi_features
    .groupby(["group", "subject_id", "region"], as_index=False)
    .mean(numeric_only=True)
)
```

Для межгруппового сравнения используются:

```text
Mann–Whitney U test
Hedges' g
rank-biserial correlation
Benjamini–Hochberg FDR correction
```

В итоговые статистические таблицы включаются:

```text
среднее значение
медиана
IQR
разность средних
разность медиан
Hedges' g
rank-biserial correlation
p-value
q-FDR
число наблюдений в группах
```

## Важные методические замечания

### Subject-level анализ

Основная статистическая интерпретация должна выполняться после агрегации на уровне субъектов. Это снижает риск псевдорепликации, когда один субъект с несколькими записями получает больший вес в статистике.

### Ограничение осей на графиках

Некоторые графики используют ограничение оси Y по высокому перцентилю для повышения читаемости при наличии выбросов.

Это ограничение применяется только к визуализации. Статистический анализ выполняется на полном наборе значений без удаления выбросов.

Рекомендуемая подпись для таких рисунков:

```text
Для повышения читаемости ось Y ограничена по высокому перцентилю; статистический анализ выполнялся на полном наборе значений без удаления выбросов.
```

### Апериодическая компонента

Апериодическая компонента оценивается с помощью линейной аппроксимации log-log PSD. В тексте не следует называть этот анализ FOOOF/specparam, если соответствующий пакет не используется явно.

## Краткое описание

Этот ноутбук реализует воспроизводимый спектральный анализ ЭЭГ для сравнения пациентов с ЧМТ и контрольной группы. Он включает расчёт PSD методом multitaper, оценку мощности частотных диапазонов, slow/fast индексов, характеристик формы спектра, апериодической 1/f-компоненты, alpha peak, residual spectrum, subject-level статистики и визуализаций для отчёта и репозитория.
