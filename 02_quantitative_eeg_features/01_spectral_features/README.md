# Spectral features output examples

This folder contains representative outputs generated by the spectral feature analysis notebook.

## Contents

- `figures/` — final figures from spectral analysis.
- `tables/` — aggregate and statistical tables that can be shared.
- `tables_anonymized_samples/` — small anonymized samples of feature tables.

## Important note

Raw EEG data are not included in this repository. Full record-level and subject-level feature tables are not included either, because they may contain internal subject or record identifiers. Only aggregate statistics and small anonymized examples are provided.

## Analysis summary

The spectral notebook includes:

1. multitaper PSD estimation;
2. absolute and relative band power;
3. slow/fast spectral indices;
4. SEF50, SEF95, spectral entropy and spectral flatness;
5. aperiodic 1/f approximation;
6. alpha peak features;
7. residual spectrum analysis;
8. subject-level group statistics with FDR correction;
9. channel-level topographic visualizations.
