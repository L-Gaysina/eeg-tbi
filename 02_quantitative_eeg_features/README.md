# Количественные ЭЭГ-признаки

Этот раздел соответствует второму этапу проекта и содержит код, таблицы и визуализации для расчёта количественных признаков ЭЭГ у пациентов с ЧМТ и контрольной группы.

## Место в проекте

1. [`01_preprocessing/`](../01_preprocessing/) — предобработка ЭЭГ и контроль качества.
2. [`02_quantitative_eeg_features/`](../02_quantitative_eeg_features/) — расчёт количественных признаков ЭЭГ.
3. [`03_machine_learning/`](../03_machine_learning/) — модели классификации и оценка информативности признаков.

На вход этого этапа подаётся очищенный inventory после предобработки: `analysis_ready_inventory_cleaned.csv`.

## Подразделы

### [`01_spectral_features/`](./01_spectral_features/)

**Спектральный анализ ЭЭГ:**

* оценка спектральной плотности мощности (power spectral density, PSD) методом multitaper;
* расчёт абсолютной и относительной мощности стандартных частотных диапазонов ЭЭГ;
* расчёт индексов спектрального замедления;
* оценка спектральной граничной частоты SEF50 и SEF95;
* расчёт нормированной спектральной энтропии и спектральной плоскостности;
* оценка апериодической 1/f-компоненты спектра;
* анализ индивидуального α-пика;
* анализ остаточного спектра после удаления апериодической компоненты;
* статистическое сравнение групп на уровне субъектов с коррекцией на множественные сравнения методом FDR;
* построение топографических карт распределения спектральных эффектов.

### [`02_temporal_connectivity_events/`](./02_temporal_connectivity_events/)

**Временные, сетевые и событийные признаки ЭЭГ:**

* расчёт DFA-подобных флуктуационных признаков амплитудной огибающей;
* анализ burst-характеристик амплитудной огибающей;
* оценка функциональной связности ЭЭГ;
* построение матриц связности между областями интереса ROI×ROI;
* расчёт графовых метрик сетевой организации;
* анализ морфологических и пространственных характеристик транзиентных событий ЭЭГ;
* статистическое сравнение групп на уровне субъектов с коррекцией на множественные сравнения методом FDR.

## Термины и обозначения

| Обозначение | Расшифровка |
|---|---|
| PSD | Спектральная плотность мощности |
| Multitaper | Многоконный метод спектральной оценки |
| SEF50 / SEF95 | Спектральная граничная частота, ниже которой сосредоточено 50% / 95% мощности спектра |
| Spectral entropy | Нормированная спектральная энтропия |
| Spectral flatness | Спектральная плоскостность |
| 1/f-компонента | Апериодическая компонента спектра |
| Alpha peak | Индивидуальный α-пик |
| Residual spectrum | Остаточный спектр после удаления апериодической компоненты |
| DFA | Анализ детрендированных флуктуаций |
| Burst | Кратковременный эпизод повышенной амплитуды огибающей |
| Functional connectivity | Функциональная связность |
| Graph metrics | Графовые метрики сетевой организации |
| ROI | Область интереса |
| FDR | Коррекция на долю ложных открытий |
